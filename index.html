Exportar dados

Backup completo do sistema unificado

Exporta agenda, pacientes, pacotes, serviços, lembretes, prontuário, sessões clínicas, evoluções, documentos, guias e configurações em um único arquivo JSON.

Recomendação FEMIC: faça backup antes de grandes alterações, antes de limpar banco e ao final de cada mês.
Restaurar dados

Restaurar backup unificado ou legado

A restauração aceita backup unificado, backup antigo da agenda e backup antigo do sistema clínico. Pacientes são sempre inseridos ou atualizados; o prontuário local é sincronizado junto com a agenda quando houver dados clínicos no arquivo.

Segurança: quando o arquivo contiver apenas dados clínicos, a agenda não será apagada. Para pesquisar anos antigos sem ocupar o Supabase, use "Consultar backup antigo".
